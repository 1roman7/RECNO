# История изменений (Changelog) - RECNO Panel

Здесь вы можете отслеживать развитие панели RECNO. При команде `recno update` система будет показывать новые изменения.

## [v1.0.0] - 2024-05-02
### Добавлено (Features)
- 🚀 **Полноценная архитектура Master-Node**: Ноды устанавливаются изолированно (в папку `/opt/recno`), работают на собственных портах (6020 gRPC) и не конфликтуют с уже существующим Marzban.
- 🌍 **Subscription Web UI (Клиентский дашборд)**: При открытии Sub Link в браузере показывается страница с круговой диаграммой трафика, датой истечения, QR-кодом и кнопками One-Click импорта в (Hiddify, V2rayTun, v2rayNG).
- 🔐 **Мульти-протоколы**: При создании пользователя теперь можно выбирать сразу несколько протоколов (VLESS, VMess, Trojan, Hysteria 2). Поддержка Hysteria 2 работает из коробки.
- ⚙️ **Встроенный редактор Xray-core (Hosts)**: Настройка Inbounds (Домены, SNI, Fingerprint, ALPN) напрямую из интерфейса без копания в JSON.
- ⏱ **Умные лимиты юзеров**:
  - `IP Limit`: Защита от раздачи ключей другим лицам.
  - `Reset Strategy`: Автоматическое обнуление трафика юзерам каждый день, неделю, месяц или год с помощью фоновых задач.
- 🤖 **Telegram Bot**: Интеграция REST API для оповещения администратора о превышении лимитов пользователями.
- 💻 **CLI Утилита (`recno`)**: Вызов консольного меню для бекапов (базы и конфигов в tar.gz), просмотра логов, перезапуска, и обновления всей панели одной кнопкой.

### Изменено (Changed)
- **Установщики (One-Liner)**: Полностью автоматизированы bash-скрипты. Автоматически скачивают ветку с GitHub (`1roman7/RECNO`), создают `venv`, генерируют SSL (через certbot или self-signed по IP), инициализируют БД и запускают панель.
- **Интерфейс (Vue 3)**: Полностью переработанный SPA интерфейс в стиле Marzban. Поддержка русского (RU) и английского (EN) языков, светлой/темной темы, Toast-уведомлений и загрузочных лоадеров.
- Фоновая синхронизация: Каждые 10 минут Master-сервер "проталкивает" активных пользователей на все Node-серверы, гарантируя отказоустойчивость.

### Безопасность (Security)
- Поддержка HTTPS для веб-панели из коробки (FastAPI стартует с переданными ключами SSL).
- Закрыта уязвимость подписок (теперь используется сгенерированный UUID `sub_id`, вместо предсказуемого логина).
- Авторизация переведена на JWT токены с хэшированием паролей (bcrypt).
