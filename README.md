# RECNO Panel (Proxy Management System)

RECNO — это полнофункциональная, автономная и гибкая панель управления Xray-core с Master-Node архитектурой. Она разработана с учетом опыта работы с Marzban, но предлагает расширенную маршрутизацию, параллельную работу нод и улучшенный интерфейс.

## 🚀 Основные возможности

1. **Master-Node архитектура**: Вы можете установить панель на один сервер, а агенты (ноды) на десятки других. Управление (gRPC) происходит по порту `6020`, что полностью решает проблему конфликтов портов с уже установленными на этих серверах Marzban / Xray панелями.
2. **Абсолютная Мультиязычность**: Панель изначально создана с фокусом на русскоязычное сообщество (включая установщики и интерфейс), но имеет кнопку переключения на Английский.
3. **One-Liner установка**: Установка в одну команду. Панель сама выпустит SSL сертификат (через Certbot или Self-Signed, если используется IP) и настроит окружение.
4. **Встроенная поддержка Hysteria 2**: Master-сервер "из коробки" способен поднимать Inbounds (включая Hysteria 2 и VLESS).
5. **RECNO CLI**: Встроенная интерактивная утилита в терминале (команда `recno`) для перезапуска сервисов, просмотра логов, обновления и сброса пароля.

## 📥 Установка

### Главный сервер (Master)

Запустите команду ниже на вашем основном сервере (рекомендуется Ubuntu 22.04 / 24.04). Скрипт спросит у вас предпочитаемый язык и домен. Если домена нет, просто нажмите Enter, скрипт выпустит локальный сертификат по вашему IP.

```bash
bash <(curl -sL https://raw.githubusercontent.com/1roman7/RECNO/main/recno-cli/scripts/install.sh)
```

### Удаленные ноды (Nodes)

На сервере, который вы хотите подключить к вашей панели, запустите агент установки. **Даже если на этом сервере уже есть Marzban (порт 443), это не вызовет конфликта**, RECNO установит изолированный бинарник.

```bash
bash <(curl -sL https://raw.githubusercontent.com/1roman7/RECNO/main/recno-cli/scripts/install_node.sh)
```

В админ-панели Master сервера просто добавьте новую Ноду, указав IP адрес этого сервера и порт (по умолчанию 6020).

## 💻 Управление (CLI)

После установки на любом сервере (Master или Node) доступна команда `recno`.
Просто введите в терминале:

```bash
recno
```
Откроется графическое меню терминала, где вы сможете:
* Посмотреть статус (Панель / Xray)
* Перезапустить демоны
* Просмотреть Live логи
* Сбросить пароль администратора (admin / admin)
* Обновить панель или ядро Xray-core
* Удалить RECNO

Вы также можете использовать аргументы: `recno restart`, `recno status`, `recno uninstall`.

## 🛠 Технический стек

* **Backend**: Python 3 (FastAPI, SQLAlchemy, SQLite, pyjwt, bcrypt, grpcio).
* **Frontend**: SPA на Vue 3 + Tailwind CSS.
* **Proxy**: Оригинальный XTLS/Xray-core (связь по gRPC Protobuf).

## 🔐 Настройка подписок и ссылок
Подписки генерируются динамически. Пользователь получает ссылку формата:
`https://ваш-домен/sub/имя_пользователя`
Панель создает "фейковые" vless-ноды в самом верху списка пользователя (в V2RayNG/Nekobox), через которые передает статус трафика, лимиты, информацию о серверах и предупреждения, полностью на Русском языке.
